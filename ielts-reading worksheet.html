What's Inside

이 학습지에 담긴 것들

단순한 해석이 아닙니다. 혼자 공부해도 강의 없이 이해되도록 설계했습니다.

01
📖
단락별 완전 분해
각 지문을 단락 단위로 나누어 집중 학습
  • 단어 정리 (품사·예문·한국어 뜻)
  • 핵심 표현 정리
  • 긴 문장 구조 분석 (청킹 + 문법 포인트)
  • 단어 매칭 / 빈칸 채우기 연습
02
🎯
키워드 기반 Answer Key
정답을 외우는 게 아니라, 찾는 법을 배웁니다
  • 문제 키워드 → 지문 키워드 매칭
  • 근거 문장 (완전한 문장)
  • 볼드 = 키워드, 색상 = 정답 표시
  • 난이도 높은 함정 문제 별도 해설
03
🔄
동의어 / 유의어 표
IELTS가 즐겨 쓰는 paraphrase 패턴 총정리
  • 문제 표현 ↔ 지문 표현 대조
  • TRUE/FALSE/YES/NO 근거 반대 표현
  • 테스트별 색상 구분 정리
  • 단순 키워드 매칭 아닌 진짜 동의어만
